HTML

  • The Head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • HTML is one of three cornerstone languages used in web dev
  • elements are used to divide a page and to decribe content to browsers and devs
  • a basic page set up in html would start with !DOCTYPE!, then HTMl lang="en", then head. inside the head you have meta and title.
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an elemnt
  • Declarations contain two important components: the CSS property we want to apply and the value of the property

  • git commit -m "description of what you changed to your code" : commits the changes
  • git push: pushes everything to your repo
